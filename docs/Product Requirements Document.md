# Product Requirements Document

## Overview
The platform must diagnose common enterprise Windows endpoint networking issues using deterministic rules and evidence.

## Functional Requirements
- Collect endpoint evidence from system data and diagnostics
- Classify issues into known categories
- Produce human-readable explanations with evidence
- Recommend remediation steps
- Support offline execution as a portable application

## Non-Functional Requirements
- Maintainable and modular
- Secure and read-only by default
- Testable with automated regression tests
- Extensible for plugin-based diagnostics
