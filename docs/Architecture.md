# Architecture

## Layers
1. Evidence Collection Layer
   - Collects endpoint and network evidence
2. Rule Engine Layer
   - Applies deterministic rules to evidence
3. Explanation Layer
   - Generates findings, explanations, and next steps
4. Reporting Layer
   - Produces structured reports for engineers

## Design Principles
- Offline first
- Modular components
- Deterministic logic
- Plugin-ready architecture
