\# PROJECT

Create a complete Enterprise Windows Endpoint Diagnostic Platform.

This is NOT a command launcher.

This is NOT a collection of PowerShell scripts.

This is a Digital Network Mentor.

The objective is to preserve senior engineer knowledge and make every new field engineer capable of diagnosing enterprise Windows networking issues quickly while learning from every incident.

\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\-\--

\# BUSINESS PROBLEM

Large enterprise organizations recruit new field engineers frequently.

There is not enough senior engineering time to teach every new engineer enterprise troubleshooting.

Engineers spend significant time manually running Windows commands and interpreting outputs.

Troubleshooting quality varies from engineer to engineer.

Knowledge leaves the company when experienced engineers resign.

The company needs a portable offline Windows application that:

• Diagnoses enterprise endpoint networking problems

• Explains WHY each issue occurred

• Shows supporting evidence

• Teaches junior engineers

• Generates professional reports

• Standardizes troubleshooting
